# LOL Chat Timer

Overwolf plugin to track spell and ability by in game chat

## Features

## Requirements

- Node version >=18

## Getting started

```sh
# install dependencies
npm i

# build overwolf plugin, you can now use `Load unpacked extension` to test the local plugin at `build`.
npm run build

# create opk file
npm run create-opk
```
